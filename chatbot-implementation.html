Phase 1: Meta WhatsApp Setup

‚è±Ô∏è 2 hours üí∞ Free

Step 1: Create Meta Developer Account

  1. Go to developers.facebook.com
  2. Click "My Apps" ‚Üí "Create App"
  3. Select app type: "Business"
  4. Fill in:
    • App Name: "Aurora WhatsApp Bot"
    • App Contact Email: your email
    • Business Portfolio: Select/create your business
  5. Click "Create App"

Step 2: Add WhatsApp Product

  1. In app dashboard, find "WhatsApp" in products list
  2. Click "Set up"
  3. Navigate to WhatsApp ‚Üí "Getting Started"

Step 3: Collect Credentials

Important: Copy these values exactly - you'll need them for Laravel .env file
Required Credentials 
1. WHATSAPP_API_TOKEN
   Location: WhatsApp > Getting Started > "Temporary access token"
   Copy: The long token string

2. WHATSAPP_PHONE_NUMBER_ID
   Location: Below the test number
   Format: 123456789012345

3. WHATSAPP_BUSINESS_ACCOUNT_ID
   Location: WhatsApp > API Setup
   Copy: Your WABA ID

4. WHATSAPP_APP_ID
   Location: Settings > Basic
   Copy: App ID

5. WHATSAPP_APP_SECRET
   Location: Settings > Basic > "Show"
   Copy: App Secret

6. WHATSAPP_WEBHOOK_VERIFY_TOKEN
   Create: Any random secure string
   Example: aurora_webhook_secure_2024

Step 4: Test WhatsApp Connection

  1. In WhatsApp > Getting Started
  2. Click "Add phone number" under "To"
  3. Enter your WhatsApp number: +593XXXXXXXXX
  4. Send the code shown to Meta's test number
  5. You should receive "Hello World" message
‚úÖ Success: If you received the message, WhatsApp setup is working!
Note: Webhook configuration comes later in Phase 7 after Laravel is set up

Phase 2: Groq AI Setup

‚è±Ô∏è 30 minutes üí∞ Free Forever

Step 1: Create Groq Account

  1. Go to console.groq.com
  2. Sign up with email or GitHub
  3. Verify your email
  4. Login to dashboard

Step 2: Generate API Key

  1. In Groq console, go to "API Keys"
  2. Click "Create API Key"
  3. Name it: "Aurora WhatsApp Bot"
  4. Copy the key (starts with gsk_...)
‚ö†Ô∏è Important: Save the key immediately! You can't see it again after closing.

Step 3: Test Groq API

Quick test to verify it works:

Terminal 
curl "https://api.groq.com/openai/v1/chat/completions" \
  -X POST \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_GROQ_API_KEY" \
  -d '{
    "model": "llama-3.3-70b-versatile",
    "messages": [{"role": "user", "content": "Hello!"}],
    "temperature": 0.7,
    "max_tokens": 100
  }'
‚úÖ Success: If you get JSON response with AI's reply, Groq is working!

Add to .env File

.env 
GROQ_API_KEY=gsk_your_key_here
